# Ludum Dare 36

## Initialization

### Install Animagic
1. ```git clone git@github.com:Kenoshen/animagic.git```
2. ```cd animagic/```
3. ```mvn install```

### Install Jump
1. ```git clone git@github.com:bitDecayGames/Jump.git```
2. ```cd jump/```
3. ```mvn install```

### Setup this project
1. ```git clone git@github.com:bitDecayGames/LudumDare36.git```
2. ```cd ludumdare36/```
3. ```mvn compile```
